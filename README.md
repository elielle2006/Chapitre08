ce projet parle de l'utilisation de tests unitaires avec JUnit5 pour vérifier le comportement d’une classe Java simple appelée BookShelf .
la classe BookShelf Stocke une liste de livres sous forme de chaînes de caractères.
les methodes : add(String... booksToAdd) : ajoute un ou plusieurs livres à la liste et books() : retourne la liste des livres présents sur l’étagère.
L’étagère est vide lorsqu’aucun livre n’est ajouté.
L’ajout de deux livres fonctionne correctement.
L’appel à add() sans argument ne modifie pas la liste.
